# 第1条（目的）

本規約は、Androidクローズドテスト攻略組（以下「本コミュニティ」という）の権利と責任を表明したものである。

# 第2条（定義）

本規約において使用する用語の定義は、次のとおりとする。

1. **「運営者」**とは、本コミュニティを運営する者をいう。
2. **「利用者」**とは、本コミュニティのサービスを利用する全ての者をいう。
3. **「本サービス」**とは、本コミュニティが提供する一切のサービスをいう。
4. **「コンテンツ」**とは、本コミュニティまたは利用者が投稿、送信、またはアップロードする一切の情報（アプリケーション、テキスト、画像、動画等を含むがこれに限られない）をいう。

# 第3条（利用者の権利）

1. 利用者は、本サービスを無償で利用することができる。
2. 利用者は、自ら開発したアプリケーションに関する一切の知的財産権を保有する。運営者は、利用者が開発したアプリケーションについて、いかなる権利も主張しない。（第10条 参照）
3. 利用者は、本コミュニティが公開する記事その他のコンテンツを閲覧し、活用することができる。

# 第4条（利用者の義務）

利用者は、本サービスの利用にあたり、次の各号に定める義務を負う。

1. 他の利用者および運営者に対し、礼節をもって接し、丁寧な言葉遣いを心掛けること
2. インターネット上の一般的なモラルおよびマナーを遵守すること
3. 他の利用者と相互に協力すること
4. 禁止事項に抵触しないこと（第5条 参照）
5. 本コミュニティまたは運営者の名誉もしくは信用を毀損し、または本コミュニティに不利益をもたらす行為を行わないこと
6. 本コミュニティ内において、運営者の合理的な指示に従うこと

# 第5条（禁止事項）

利用者は、本サービスの利用にあたり、次の各号のいずれかに該当する行為を行ってはならない。

1. 他の利用者もしくは運営者または第三者の権利もしくは利益を侵害する行為
2. コンピュータウィルスその他の有害なプログラムを使用または提供する行為
3. その他、運営者が不適切と判断する行為

# 第6条（運営者の権利）

1. 運営者は、利用者が本サービスを通じて製品版として完成させたアプリケーションを、本コミュニティ外で紹介する権利を有する。
2. 運営者は、本コミュニティおよび本サービスに関する一切の権利を保有する。（第10条 参照）

# 第7条（運営者の義務）

運営者は、次の各号に定める義務を負う。

1. 利用者に対し、礼節をもって接し、丁寧な言葉遣いを心掛けること
2. インターネット上の一般的なモラルおよびマナーを遵守すること
3. 寄付を除き、本サービスを無償で運営すること

# 第8条（利用制限および登録抹消）

1. 運営者は、利用者が次の各号のいずれかに該当する場合、事前の通知または催告なしに、当該利用者による本サービスの利用を制限し、または本コミュニティから除外することができる。
    1. 本規約のいずれかの条項に違反した場合
    2. その他、運営者が本サービスの利用を不適切と判断した場合
2. 前項の措置に関し、運営者は利用者に対して理由を開示する義務を負わず、また、当該措置に起因して利用者に生じた損害について一切の責任を負わない。

# 第9条（コンテンツの投稿および情報発信）

本コミュニティは、法的な法人格および個人格を持たない。このため運営者と利用者および第三者の安全のため、外部サービスを通してのみコンテンツの投稿および情報発信を行う。

運営者は、外部サービスとして、X（旧Twitter）、Instagram、Zenn、Qiita、noteその他のSNSまたはブログ投稿サイトを利用するものとし、各外部サービスの利用規約を遵守する。（第10条 参照）

# 第10条（知的財産権）

1. 本コミュニティおよび本サービスに関する知的財産権は、全て運営者に帰属する。ただし、利用者が開発したアプリケーションに関する知的財産権は、当該利用者に帰属するものとし、この限りではない。
2. 運営者は、外部サービスにコンテンツを投稿するにあたり、第三者の知的財産権を侵害しないよう細心の注意を払うものとする。
3. 第三者が、運営者の投稿したコンテンツが自己の知的財産権を侵害すると主張する場合、当該第三者は、当該外部サービスの運営事業者が定める手続に従い、当該運営事業者に対して報告を行うものとする。運営者は、当該運営事業者からの連絡を受けた場合、速やかに対応するものとする。（以下のリストを参照）
4. 運営者は、当該外部サービスの運営事業者が定める手続を経由しない知的財産権侵害の申立てに対し、直接的な紛争やトラブルに発展しない範囲で回答することができる。ただし、回答の義務は負わず、最終的な判断・措置は当該外部サービスの運営事業者の裁量に委ねるものとする。
